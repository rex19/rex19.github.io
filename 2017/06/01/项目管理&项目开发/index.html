发表于 2017-06-01

工厂车间现场Reporting信息化项目

实施地点及时间：

重庆铜梁**汽车零配件工厂 2016/9

项目角色：

项目经理,产品设计,开发/实施工程师

项目背景：

工厂上线Fanuc机器人，机床自动化一体的新产线，同时上线Reporting信息化系统

软件主要功能：

Andon系统部分,监控机器人,机床各种实时状态，报警信息,及时邮件通知给工程师,实时显示每个产线,每个班次的产量;
生产操作工录入每个班次的生产信息，不良品信息;
自动计算产线每天,每周,每月的Cycletime,Downtime,计划完成率,产品合格率,时间稼动率/性能稼动率,OEE等信息,生成各种图表,表格形式反馈给产线主管;
后台管理系统形成可正追溯/反追溯表格,人性化历史查询功能

项目开发技术栈&架构：

Bootstrap+jQuery+ejs+HighCharts+Node.js(ES6)+Express.js+Sequelize(ORM)+Mysql+PM2(Linux) <== Kingview (OpcServer) ==> PLC(S7-1215c)

办公室智能化系统

实施地点及时间：

公司内部项目 2016/12

项目角色：

设计,前端开发

项目说明：

公司内部自主开发窗户/窗帘控制, 温湿度/二氧化碳/光照度监控

监控办公室内部温度,湿度,二氧化碳,光照亮度等数据,以图表形式展示;
反控窗户/窗帘;

项目开发技术栈&架构：

ReactNative+Antd-mobile(UI)+Echarts +C# <==> PLC(Delta) <==> Device

项目管理系统

实施地点及时间：

公司内部项目 2017/2

项目角色：

设计,前端开发

项目说明：

公司项目越来越杂,且总经理不能随时随地总览各个项目的进度,遂内部开发一套项目管理软件

软件主要功能：

跟踪各个项目的实时情况,以表格和图表形式展示;
按照公司组织架构,总经理,项目经理,自动化/信息化研发,工程师等不同级别可以分配以不同的权限;
项目经理将其做好的Project 进度上传到系统中,自动生成进度提示;
项目需要的软/硬件大额采购需求 由项目经理通过此系统向总经理提交申请;
公司每个工程师在什么时间段参与进什么项目,什么时间需要项目出差,一目了然;

项目开发技术栈&架构：

React.js+Router+Redux+AntD(UI)

上海Fanuc机器人公司工业焊接SmartLink系统

实施地点及时间：

**汽车行业公司&2017届北京埃森焊接与切割展览会 2017/6

项目角色：

设计,开发/实施工程师

项目背景：

Fanuc机器人要向其客户推广自动/信息一体化方案,我司与日本一上市公司B-en-g竞争,后成功取得客户最终认可,在Fanuc埃森展上做信息化系统展览销售

软件主要功能：

采集客户集团化的云端数据，可以看到整个集团的生产情况(包括质量合格率,计划完成率,综合效率等);且可触屏点击地图下钻到中国区每个省份/城市的工厂数据(客户高管可以根据web界面的地图选择不同工厂) ;
车间看板层：
可以看到每个工厂实时的生产信息,包括每个焊接工作站的机器人状态(Running? Ready? Fault? PowerOff..)及产量/合格率/停机率/计划完成率/安全生产天数等信息;
模态框触屏方式进去 可以看到每个机器人的开动率,运行状态比率,故障类型比率,生产产品类型比率等饼图, 每天每小时每周的产量/停机时间 柱形图等..
再次下钻到机器人具体的报警信息,实时采集的机器人焊接参数/焊接电流电压的波形图;
追溯查询层：
有车间看板层可下钻到后台追溯系统, 可以查询到每个工件号对应的焊缝参数信息,生产时对应的机器人,焊机等信息,同时也可以反追溯
App:
ReactNative 开发Android Apk, 使客户工厂领导可以在手机端实时查询到若干工厂内部生产信息

项目开发技术栈&架构：

React.js(PC-End)+Webpack+ReactNative(Mobile-End)+Semantic-UI+AntD(UI)+Node.js+Express.js+Sequelize+MongoDB +Ngnix(Windows) <== KepWare(OpcServer) ==> PLC(AB)
