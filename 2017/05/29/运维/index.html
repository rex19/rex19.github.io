创业公司运维部分


2016年3月-8月，除了软件开发方面工作外，帮助公司从无到有建立IT基础设施
包括网络，电话，服务器，电脑，ERP/OA等部分

网络/门禁/电话

宽带部署，选型，采购并配置企业路由器，交换机，AP， 网线等耗材，完成部署
部署人脸识别/指纹门禁系统
电话选型，采购并配置电话程控交换机，电话机，完成布线等操作

服务器/电脑

选型，采购并配置 公司的文件服务器(NAS),应用服务器(WinServer2012)
参与设计公司组织架构，部门权限，并映射到服务器的权限管控
搭建公司应用服务器的各个虚拟服务器，包括项目/业务中不同应用场景用到的Linux/Win虚拟服务器
搭建公司邮件服务器
电脑选型选型并采购公司内部工作使用电脑， 包括研发/业务等不同工作需要用到不同电脑配置等

ERP/OA/其他应用软件

选型，采购并配置一系列公司业务/研发需要用到的软件，如AutoCAD，Office,Solidworks,Proe 等。并持续维护License
部署阿里钉钉作为内部流程，通讯软件
设计并主导开发 公司内部OA审批应用 包括人事模块，采购模块，财务模块等，上线后持续优化直到稳定使用
选购,部署 ERP => 用友U8 系统上线, 连接 采购-物流-仓库-财务 企业资源管理一体化业务流程方案信息化
软件开发方面代码版本控制 Git

项目管理&项目开发
